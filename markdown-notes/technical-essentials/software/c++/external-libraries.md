## pagmo2

- A framework for solving difficult optimization problems
- Has various algorithms to optimize and tune a system
- NSGA-II
  - "non-dominated sorting genetic algorithm II"
  - Optimizes for conflicting objectives at the same time using algorithms inspired by biological evolution
  - Generates a pareto-front of solutions that balance different variables to maximize performance
