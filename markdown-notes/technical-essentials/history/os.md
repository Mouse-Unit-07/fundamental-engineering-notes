## A Rough Timeline

- **GM-NAA I/O**
  - 1950s
  - First OS for IBM 704
- **Compatible Time-Sharing System (CTSS)**
  - Time-Sharing OS at MIT
- **IBM OS/360**
  - General purpose OS for System/360
  - Standardized business and scientific computing
- **MULTICS**
  - Inspired Linux
  - Hierarchical file systems, dynamic linking, etc
- **UNIX**
  - Ken Thompson and Dennis Ritchie
  - Portable, multiuser, multitasking
  - Foundation for other OS's
- **PDP-11 / RT-11**
  - DEC's RTOS for minicomputers
- **UNIX rewritten in C**
  - UNIX made further portable
- **Apple DOS 3.1**
  - Disk-based OS for Apple II
- **MS-DOS 1.0**
  - Microsoft
  - Shipped on IBM PCs
- **Apple Lisa OS**
  - First GUI-based OS
  - Windows, icons, menus
- **Mac OS (System 1)**
  - User-friendly GUI OS for Macintosh
- **Windows 1.0**
  - Microsoft's first GUI layer on top of MS-DOS
- **MINIX**
  - UNIX-like OS for teaching
  - Also inspired Linux
- **NeXTSTEP**
  - Unix-based OS w/ object-oriented concepts
  - Led to macOS and iOS
- **Linux kernel 0.01**
  - Open-source UNIX-like OS
- **Windows 3.1**
  - PCs made mainstream for business and homes
- **Windows NT 3.1**
  - 32-bit architecture- foundation for all Windows versions
- **Windows 95**
  - Start menu, 32-bit hybrid kernel, plug and play
- **Palm OS**
  - Handheld OS, led to smartphone OS's
- **Windows 2000 / XP**
  - Unified consumer and enterprise OS line
  - Wintel became the industry standard
- **Mac OS X 10.0**
  - UNIX-based rewrite of macOS
- **iOS 1.0**
  - Apple's mobile OS for iphone
- **Android 1.0**
  - OS for Android devices
- **Windows 8**
  - 2012
  - Touch oriented redesign for tablets and PCs
- ...
