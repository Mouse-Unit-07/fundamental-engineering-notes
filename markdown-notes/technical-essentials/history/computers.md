## 5 Generations

- The notable "first to do xxx"'s across computer development history
- **0th Generation (1642-1945): Mechanical Computers**
  - Calculator birth
    - Blaise Pascalâ€™s Pascaline
    - Gears to add/subtract
  - General purpose calculator
    - Charles Babbage
    - Difference engine
      - Performed Polynomial calculation
    - Analytical engine
      - General purpose programmable calculator
      - Augusta Ada Lovelace
        - The first programmer- wrote an algorithm for the analytical engine
  - Programmable digital calculator w/ electromagnetic relays
    - Konrad Zuse
  - Digital calculator w/ binary and capacitors
    - John Anasoff, Clifford Berry
  - Electromechanical relay based digital calculator
    - George Stibitz at Bell Labs
  - ENIGMA, COLOSSUS, ENIAC: WWII
    - ENIGMA
      - Machine and encryption used by Nazis, broken by cryptanalysts
    - COLOSSUS
      - British code breaking machine w/ vacuum tubes
    - ENIAC
      - Large-scale general-purpose electronic computer
  - John von Neumann architecture
    - Stored program design where instructions and data are in the same memory
    - Basis for modern computers
  - Digital computer patent attempt
    - Eckert & Mauchly tried to patent the invention of the digital computer, but became public domain thanks to Atanasoff's work
- **1st Generation (1945-1955): Vacuum Tubes**
  - IBM
    - Dominated w/ punch card machines and then vacuum-tube computers
  - FORTRAN Monitor System
    - An operating system, running FORTRAN the high-level programming language by IBM
  - "Microgramming"
    - Control of CPU via microinstructions stored in control memory
  - Scientific computer
    - IBM 701
- **2nd Generation (1955-1965): Transistors**
  - Transistor invention
    - John Bardeen, Walter Brattain, and William Shockley at Bell Labs
    - Replaces vacuum tubes
  - DEC PDP-1
    - Commercial minicomputer- nice and affordable
  - MIT real-time minicomputers
    - Jay Forrester's Whirlwind further pushed minicomputers
  - IBM System/360
    - Unified scientific and business computing
    - Multiprogramming
    - Backwards compatibility as opposed to machines that don't integrate well together
  - CDC 6600
    - Seymour Cray
    - A supercomputer that beat IBM's 7094
  - Burroughs B5000
    - A stack-based machine that emphasized software/compiler design as a key to computer architecture
- **3rd Generation (1965-1980): Integrated Circuits**
  - IC invention
    - Jack Kilby & Robert Noyce
  - Ralph Baer's Magnavox Odyssey
    - Birth of video games, starting w/ Pong
  - IBM 1401
    - Cheaper machines for businesses to use for computing
  - MITS Altair 8800
    - Start of a personal computer revolution
  - Apple
    - Apple I, II kick off as microcomputers
  - RISC vs CISC
    - Engineers argue over whether RISC or CISC is better, resulting to the mix we have now
- **4th Generation (1980-xx): Very Large Scale Integration**
  - VLSI
    - Thousands of transistors per IC
  - FPGA
    - Ross Freeman at Xilinx
    - Reconfigurable logic devices invented
  - IBM personal computer
    - Open architecture, ran MS-DOS licensed from Microsoft
  - Osborne-1
    - Portable computer
  - Commodore 64
    - Successful home computer
  - Apple Lisa
    - Commercial computer w/ a GUI
  - Cray-1, -2
    - Seymour Cray
    - Pushes supercomputing
  - Japan's 5th generation project
    - Declares $500 million investment in AI-focused computers
    - ...ends in a flop lmao
  - ARM architecture
    - Was originally "Acorn RISC Machine" lol
  - Intel & Microsoft dethrone IBM
    - Windows + Intel CPUs very powerful
  - X86 architecture
    - Becomes an industry standard for PCs/servers
  - Windows 95
    - Mass adoption of personal computers w/ GUI
  - MIPS R4000, DEC Alpha
    - 64-bit CPUs
    - Workstation/server-class CPUs
  - IBM POWER4
    - Multiprocessor w/ 2 cores on a single die
- **5th Generation ("Now"): Low-power and Invisible Computers**
  - Spread of embedded computing
    - MCUs and embedded ICs across various applications
  - Smartphone, ARM dominates mobile
  - Billions of transistors per IC
  - X86 vs ARM rivalry, RISC-V rising, FPGAs used in datacenters, AI accelerators, GPUs

## 80 Characters

- Back in the day, terminals were cathode ray tubes that displayed 24 lines by 80 columns
- That's it- that arbitrary 80 is why we frequently go by 80 characters per line

## Binary Representation

- ...We've come a long way to get to two's complement, floating point, etc that we have now
- Morse Code
  - Invented by Samuel Finley Breese Morse in 1837
  - It was built on later by others, but Morse code w/ its dots and lines is binary
- Braille
  - Louis Braille invented Braille in 1824
  - The presence/absence of dots is also binary

## Byte

- IBM coined the term "byte" in 1956
- It originally meant the number of bits in a datapath, but was changed to mean 8 bits

## Keyboard Keys

- Shift key
  - "Shift/precedence codes" refers to codes alter the meaning of any following codes until the shift is done
  - There was a "shift" key on old typewriters, and they haven't gone away
- Escape key
  - "Escape codes" refer to codes that escape from some normal interpretation
  - Braille implemented this to indicate that an escaped character should be uppercase rather than lowercase
- Carriage return / line feed
  - Carriage return
    - Refers to the way typewriters had carriages holding paper return to the right side after finishing a line
  - Line feed
    - Refers to the way the carriage on a typewriter is rolled so that you start writing on the next line after finishing a line
- Tab
  - Stands for "horizontal tabulation code" lol
  - It was originally meant to print the next character at the next horizontal position that's a multiple of 8
